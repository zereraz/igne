- Press `Cmd+P` for command palette
- Try prefix modes:

  - `#` for tags
  - `:` for symbols
- Press `Cmd+F` for find/replace

### 8. **File Management**
- Right-click file tree for options
- Rename file (links update automatically)
- Star important files
