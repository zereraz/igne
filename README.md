# Igne

A fast, native markdown editor. Opens your Obsidian vaults.

## Download

**[Download v0.3.0](https://github.com/zereraz/igne/releases/latest)**

- macOS (Apple Silicon / Intel)
- Windows
- Linux

## Why Igne?

- **Fast** — Native app, instant startup
- **Familiar** — Works with your existing Obsidian vault
- **Simple** — No electron, no bloat

## Features

- Markdown editor with live preview
- Wikilinks `[[like this]]`
- Daily notes & templates
- Quick switcher (Cmd+P)
- Graph view
- Backlinks
- Math (KaTeX) & diagrams (Mermaid)
- Light/dark themes
- Community themes & CSS snippets
